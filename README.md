# landing-page
Responsive landing page built during my front-end learning journey using **HTML, CSS, SASS**

## Project Overview
This project was created as part of my front-end practice to improve my skills in:
- **HTML** - semantic structure and accessibility
- **CSS** - styling, layout, responsiveness
- **Flexbox & Grid** - modern layout techniques
- **Media Queries** - mobile-first approach for responsive design


## Technologies Used 
- **HTML5** - for content structure
- **CSS3/SASS** - for styling
- **Flexbox & Grid** - for layout control
- **Responsive Design** - works on mobile, tablet and desktop

## Notes
This project reflects my learning process. All adjustments, spacing fixes, responsive inprovements and minor UI polish were added progressively showing a workflow in git commits.
